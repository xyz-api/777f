# 酒店账单核对使用指南

这个工具用来对比酒店账单和入住登记表，找出哪些账单有对应的登记记录，哪些没有。

## 使用场景

你有两份表格：
- 酒店账单表：酒店给的消费明细，包含姓名、日期、金额等
- 入住登记表：内部的入住申请记录，包含姓名、入住日期等

你需要核对：账单里的每一条消费，是否都有对应的入住申请。

## 使用步骤

### 第一步：上传文件

分别上传账单表和登记表（Excel格式）。

如果表格的表头不在第一行，需要选择正确的表头行。比如表头在第二行，就选"表头在第2行"。

### 第二步：配置账单表

上传后会显示表格预览。你需要告诉工具：
- 哪一列是姓名
- 哪一列是日期

然后勾选你想在结果中显示的列。

### 第三步：配置登记表

同样选择姓名列和日期列，勾选要显示的列。

### 第四步：开始匹配

设置日期容差。因为账单日期和登记日期可能有1-2天的误差，所以需要设置一个容差范围。

比如设置"±2天"，那么账单日期是1月5日，登记日期是1月3日到1月7日之间的都算匹配。

点击"开始匹配"，工具会自动比对两张表。

### 第五步：查看结果

结果会显示每条账单的匹配状态：
- 已匹配：找到了对应的登记记录
- 未匹配：没有找到对应的登记记录

你可以点击"导出Excel"下载结果。

## 注意事项

- 姓名匹配是精确匹配，"张三"和"张 三"会被认为是不同的人
- 日期格式需要是 Excel 能识别的日期格式
- 如果一个人在同一天有多条记录，会匹配到第一条
