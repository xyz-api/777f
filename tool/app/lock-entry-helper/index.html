通用锁班助手（测试）

自动化填写飞行门户非生产任务录入表单

工具介绍

这是一个 Python 脚本工具，用于自动化填写飞行门户的非生产任务录入表单，支持批量录入锁班信息。

支持批量/手动模式、白名单过滤、冲突检测等功能。

两个版本说明：
  • app.py - 标准版：遇到冲突或错误时会暂停，等待用户选择跳过(s)、重试(r)或返回(b)
  • app2.py - 静默版：遇到冲突或错误时自动跳过并记录，不暂停，最后统一输出错误报告
