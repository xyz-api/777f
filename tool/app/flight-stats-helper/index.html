飞行经历起落数按天统计（测试）

自动化查询飞行门户飞行经历起落数并按天统计

工具介绍

这是一个 Python 脚本工具，用于自动化查询飞行门户的飞行经历起落数，并按天进行统计。

支持批量查询、Excel导入、白名单过滤等功能。

三个版本说明：
  • app.py - 标准版：基础功能版本
  • app1.py - 增强版：功能增强版本
  • app2.py - 静默版：自动处理，遇到错误不暂停，最后统一输出错误报告

使用说明

使用前请确保已安装以下依赖：

  • Python 3.7+
  • playwright（浏览器自动化）
  • openpyxl（Excel处理）
  • colorama（终端颜色）

安装命令：

pip install playwright openpyxl colorama
playwright install chromium
